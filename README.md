# Crop-Health-Detection
An AI-powered web application designed to detect crop diseases from leaf images using Deep Learning and Computer Vision. This project helps farmers and agricultural researchers identify plant diseases quickly and accurately, providing disease information, confidence scores, and prevention tips for healthier crop management.
